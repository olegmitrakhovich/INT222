HTML5 New Form Attributes
A major addition to HTML5 are the new form attributes. There are 14 in total: placeholder, autofocus, autocomplete, required, pattern, list, multiple, novalidate, formnovalidate, form, formaction, formenctype, formmethod, formtarget.

How to Set AutoFocus

Autofocus
<input type="text" name="entry01" id="entry01" autofocus="autofocus" />

How to Set a Placeholder

PlaceHolder
<input type="text" name="placeholder" placeholder="Place Holder" />

How to set required for an entry

Required
<input class="input" type="text" name="usrname" required />

How to create a list/datalist

Datalist
<input class="input" list="list" name="languages" / >
<datalist id="list">
<option value="HTML5">
<option value="JAVASCRIPT">
<option value="HTML">
<option value="JAVA">
<option value="C++">
</datalist>

How to ensure a valid email address format

Input type=Email
<input class="input" type="email" name="email" placeholder="Please Enter a Valid Email"/>

How to ensure a valid url entry

Input Type=Url
< input class="input" type="url" name="validsite" pattern="http?://.+" />
